# VISUALIZE.io - Interactive Visualizer Workspace 🚀

A highly responsive, beautiful, data-agnostic visualizer workbench equipped with robust client-side analytical metrics cards, interactive data exploration grids, and a server-side Gemini 3.5 AI insight pipeline.

Styled with the dynamic, colorful **Vibrant Palette** theme to bring visual life, balance, and deep focus to complex quantitative variables.

---

## ✨ Key Features

1. **Vibrant Palette Aesthetic**:
   - Clean, high-contrast, modern UI featuring deep indigo elements paired with fuchsia gradient accents.
   - Elegant **Plus Jakarta Sans** typography paired with **JetBrains Mono** for numerical values and technical status indicators.
   - Smooth hover micro-transitions, rounded 2xl/3xl bento card containers, and localized background glow states.

2. **Ingress Control Desk**:
   - Dynamic dataset switching with high-fidelity preloaded templates (Finance, Marketing, Fitness, SaaS metrics).
   - Real-time custom CSV, TSV, or raw JSON object string parser and validation with intuitive error tracking.
   - On-the-fly attribute mapping calibration (Continuous Numeric vs Categorical vs DateTime classification).
   - Rich multi-conditional filter rules engine to build custom slice-and-dice workflows.

3. **Dashboard Hub & Charts Viewer**:
   - Versatile layout configurations supporting Multi-Series Stacked/Grouped Bars, Smooth Areas, Line Graphs, responsive Scatter Plots, Radars, and Pies.
   - Adaptive dual-axis (Secondary Y-Axis) tracking to compare disparate scale metrics simultaneously.
   - Fully customizable options: toggle grids, responsive legends, horizontal orientations, and motion-powered enter animations.

4. **Interactive Metrics Architect**:
   - Responsive KPI top row with custom KPI card builder.
   - Instantly calculate and display Sum, Mean Average, Minimum, Maximum, or strict Count Rows aggregations over active attributes.

5. **AI Data Scientist Insight Engine**:
   - Dynamic server-side Gemini 3.5 Assistant to automatically read filtered row stats and produce structured executive bullet digests.
   - Live query box to ask interactive natural language questions about your active records.
   - Dynamic custom layout recommendations with instant configuration triggers (1-click preset bindings).

---

## 🛠️ Technology Stack & Dependencies

- **Frontend**: React 18+ powered by **Vite**
- **Styling**: **Tailwind CSS v4** with a custom high-contrast fluid theme
- **Visuals & Charts**: **Recharts** & **lucide-react** icons
- **Animations**: **motion** (by Framer)
- **AI Intelligence**: Server-side **@google/genai** SDK integrated via Node.js Express routes
